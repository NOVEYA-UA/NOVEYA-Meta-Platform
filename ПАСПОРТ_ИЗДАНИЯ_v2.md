## VI. ФОРМУЛЫ — НУЛЕВОЕ ИСКАЖЕНИЕ

### Египетский π

```
S = (8/9 × d)²
πе = 4(8/9)² = 256/81 ≈ 3,1605
```

### Разложение 2/n — основная формула

```
2/n = 1/(n/2+1/2) + 1/(n(n+1)/2)   — для нечётных n
```

Пример (из текста):
```
2/5  = 1/3 + 1/15
2/7  = 1/4 + 1/28
...  [50 разложений — набирать по оригиналу]
```

### Платиновое отношение (Гл. 9)

```
r = R × 8/9
r = (8/9)² × ρ = 64/81 × ρ
```

## VII. ПРАВИЛА РЕДАКТИРОВАНИЯ STR-ФАЙЛОВ

### Синтаксис правок

| Метка | Значение | Действие |
|---|---|---|
| `{ms46 — ...}` | Ссылка на маш. страницу | Игнорировать (служебная) |
| `{текст}` | Редакторский комментарий | **Не включать в вёрстку** |

| Текст вне `{...}` | **Авторский текст** | Включить в вёрстку |
| `#\формула\#` | Математическое выражение | Набрать формулу |
