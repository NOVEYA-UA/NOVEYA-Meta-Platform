## 📘 Инструкция по загрузке в Notion

1. Откройте рабочее пространство Notion
2. Создайте новую страницу “Грантовые заявки НОВЕЯ”
3. Импортируйте:
   - `NOVEYA_GRANTS_OVERVIEW.md` как текст
   - LOI и карту — как вложения
   - PPTX — в раздел презентаций
4. Добавьте таблицу с проектами (из Excel или Markdown)

---

## 📘 Инструкция по загрузке в GitHub

1. Создайте репозиторий `grants-noveya`
2. Загрузите следующие файлы:
   - `NOVEYA_GRANTS_OVERVIEW.md`
   - `NOVEYA_LOI_TEMPLATE.md`
   - `NOVEYA_PROJECTS_PRESENTATION.pptx`
3. Добавьте README с описанием структуры
4. При необходимости — создайте ветки под каждый проект
