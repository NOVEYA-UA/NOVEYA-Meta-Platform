
```
Код очищается от:
- Частотного шума

- Застойных процессов
```

## 🔄 ЦИКЛ NEBI-ULA

### Схема работы

```
┌─────────────────────────────────────────────┐
│  1. ТЕЗИС (T)                               │
│     └─ Фиксация состояния и запроса        │
│     └─ Классификация (strategic/tactical)  │
└─────────────────┬───────────────────────────┘
                  │
┌─────────────────▼───────────────────────────┐
│  2. АНТИТЕЗИС (A)                           │
│     └─ Проверка через БТ (Блок Требований) │
│     └─ Есть ли ресурсы? (Ср1-Ср9)          │
│     └─ Достаточна ли значимость? (Zn)      │
└─────────────────┬───────────────────────────┘
                  │
┌─────────────────▼───────────────────────────┐
│  3. СИНТЕЗ (S)                              │
│     └─ Разрешение противоречия             │
│     └─ УТВЕРЖДЕНО / ОТКЛОНЕНО / КОРРЕКЦИЯ  │
└─────────────────┬───────────────────────────┘
                  │
┌─────────────────▼───────────────────────────┐
│  4. ТЕСТ                                    │
│     └─ Проверка корректности синтеза       │
│     └─ Валидация (score 0-1)               │
└─────────────────┬───────────────────────────┘
                  │
┌─────────────────▼───────────────────────────┐
│  5. СТОП                                    │
│     └─ Завершение или новый цикл           │
│     └─ Передача в следующий блок (БИ)      │
└─────────────────────────────────────────────┘
```

## 🧪 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Пример 1: Законный запрос

```python
from nebi_ula_cycle import NebiUlaCycle

# Инициализация
cycle = NebiUlaCycle()

# Регистрация ресурсов
resources = {"Ср1": 100, "Ср2": 80, "Ср3": 90, ...}
cycle.block_requirements.register_resources(resources)

# Выполнение цикла
request = "Создать план восстановления Града-Сада"
result = cycle.execute_cycle(request, "CU1")

# Результат
print(result.synthesis.status)  # → "УТВЕРЖДЕНО"
print(result.next_block)        # → "БИ (Блок Идей)"
```

### Пример 2: Авантюрный запрос (блокировка)

```python
# Обнуление ресурса
cycle.block_requirements.resources["Ср9"] = 0

# Попытка запроса
request = "Запустить международную программу"
result = cycle.execute_cycle(request, "CU4")

# Результат
print(result.synthesis.status)    # → "ОТКЛОНЕНО"
print(result.antithesis.contradiction)  
# → "АВАНТЮРА: Отсутствуют ресурсы ['Ср9']"
```
